## 🏛️ عن المشروع

يعتبر هذا المشروع منصة رقمية متطورة تهدف إلى رقمنة الخدمات المقدمة من مكتب الأشغال العامة والطرق في محافظة ذمار. يعد المشروع نقلة نوعية في أساليب العمل الإداري والهندسي من خلال توفير بنية تحتية رقمية متكاملة تلبي احتياجات المواطنين والجهات الحكومية المعنية.

### الأهداف الاستراتيجية:
- تطوير البنية التحتية الرقمية للمؤسسات الحكومية
- رفع كفاءة الخدمات المقدمة للمواطنين
- تقليل الفترات الزمنية لمعالجة المعاملات
- ضمان الشفافية والموثوقية في الخدمات الهندسية
- توحيد المعايير الفنية والإدارية

---

## 🛠️ المواصفات التقنية

### البنية التحتية التقنية
- **إطار العمل:** React 18.2 مع TypeScript 5.3
- **أداة بناء:** Vite 5.4
- **نظام التصميم:** Tailwind CSS 3.4
- **إدارة الحزم:** pnpm

### المواصفات الوظيفية
- نظام إدارة المعاملات الإلكترونية
- نماذج هندسية رقمية (8 نماذج رسمية)
- نظام تتبع المعاملات
- تكامل مع قواعد البيانات السحابية
- واجهات برمجة التطبيقات (APIs)

## 📊 المواصفات الفنية

### هيكل النظام
```
الوحدة الأساسية للتطوير:
├── واجهة المستخدم (Frontend)
│   ├── مكونات واجهة تفاعلية
│   ├── نظام تصميم متجاوب
│   └── دعم الأجهزة المحمولة
│
├── البنية التحتية (Infrastructure)
│   ├── إدارة الحالة (State Management)
│   ├── التوجيه والتنقل (Routing)
│   └── التحقق من البيانات (Validation)
│
├── النظم المساعدة (Utils)
│   ├── أدوات الأمان
│   ├── أدوات التحقق
│   └── أدوات التكامل
│
└── النشر والنشر (Deployment)
    ├── إعدادات الإنتاج
    ├── الضغط والتحسين
    └── Service Worker
```

### المؤشرات التقنية
| المؤشر | القيمة |
|--------|---------|
| عدد الصفحات | 10+ صفحات |
| عدد النماذج | 8 نماذج هندسية |
| حجم البناء المُحسَّن | ~500KB (مضغوط) |
| درجة الأداء (Lighthouse) | 90+ |
| دعم المتصفحات | جميع المتصفحات الحديثة |

## 🔧 بيئة التطوير

### المتطلبات الأساسية
```env
Node.js: >= 18.x
pnpm: >= 8.x
Git: للإصدارات
TypeScript: 5.3
```

### خطوات النشر المحلي
```bash
# استنساخ المستودع
git clone https://github.com/Abdalmalik25/DhamarPWO.git

# تثبيت الاعتماديات
pnpm install

# إعداد متغيرات البيئة
cp .env.example .env
# تحديث القيم حسب بيئة العمل

# تشغيل الخادم المحلي
pnpm dev

# بناء الإنتاج
pnpm build
```
