## 🟠 تحسينات الأداء المطلوبة

### 1. تحميل مسبق غير فعال
**الملف:** `src/shared/hooks/usePrefetch.ts`  
**المشكلة:** 
- استخدام `useRef` بشكل مفرط
- عدم تنظيف `timeout` بشكل صحيح
- `requestIdleCallback` قد لا يعمل في جميع المتصفحات

**الحل المقترح:**
```typescript
// تحسين usePrefetch
const performPrefetch = useCallback(async () => {
  if (hasPrefetched.current || !isMounted.current) return;
  
  // تنظيف更好的
  if (abortControllerRef.current) {
    abortControllerRef.current.abort();
    abortControllerRef.current = null;
  }
  
  // استخدام timing أفضل
  const startTime = performance.now();
  setIsLoading(true);
  
  try {
    const result = prefetchFn();
    if (result instanceof Promise) {
      await result.catch(() => {});
    }
    hasPrefetched.current = true;
  } finally {
    const duration = performance.now() - startTime;
    console.log(`Prefetch completed in ${duration.toFixed(2)}ms`);
    setIsLoading(false);
  }
}, [prefetchFn]);
```
